# Contributing to Markdown Rich Preview & Export

First off, thank you for considering contributing to this project! It's people like you that make the open-source community such an amazing place.

## Code of Conduct

By participating in this project, you are expected to uphold our [Code of Conduct](CODE_OF_CONDUCT.md).

## How Can I Help?

- **Reporting Bugs**: Open an issue with a clear description of the bug and steps to reproduce it.
- **Suggesting Enhancements**: Open an issue to discuss new features or improvements.
- **Pull Requests**:
    1. Fork the repository.
    2. Create a new branch (`git checkout -b feature/amazing-feature`).
    3. Make your changes.
    4. Run tests (`npm test`) and ensure they pass.
    5. Commit your changes (`git commit -m 'Add some amazing feature'`).
    6. Push to the branch (`git push origin feature/amazing-feature`).
    7. Open a Pull Request.

## Development Setup

1. Clone your fork.
2. Install dependencies: `npm install`.
3. Open the project in VS Code.
4. Press `F5` to start a development instance of VS Code with the extension loaded.
5. Use `npm run watch` to automatically compile changes.

## Testing

Ensure all tests pass before submitting a PR:
```bash
npm test
```

## Pull Request Guidelines

- Follow the existing code style.
- Include unit tests for new features.
- Update documentation (like `example.md`) if necessary.
- Ensure the CI workflow passes.

Thank you for your contributions!
