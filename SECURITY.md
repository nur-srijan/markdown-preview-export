# Security Policy

## Supported Versions

The following versions of `markdown-rich-preview` are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not use the public issue tracker. Instead, please report it via the following methods:

1. **Email**: [nursrijan1421@gmail.com](mailto:nursrijan1421@gmail.com)
2. **GitHub Security Advisory**: Use the "Report a vulnerability" button on the [Security tab](https://github.com/nur-srijan/markdown-preview-export/security/advisories) of this repository.

Please provide a detailed description of the vulnerability and the steps required to reproduce it. We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Disclosure Policy

When a vulnerability is reported, we will:

1. Acknowledge the receipt of the report.
2. Investigate and confirm the vulnerability.
3. Work on a fix and validate it.
4. Release a new version containing the fix.
5. Publicly disclose the vulnerability once the fix is released.

Thank you for helping keep this project secure!
